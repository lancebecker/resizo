Resizo
=======
A handy little resizer script.

## Required to run
- ruby / rubygems
- homebrew
- FreeImage

## Gems
- trollop
- image_science
- RubyInline

## Dependency installation guide
Run these commands to get the dependencies

        sudo brew install FreeImage
        sudo gem install trollop image_science RubyInline
